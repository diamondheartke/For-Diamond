💎 For Diamond

A simple, clean donation web app built using HTML, CSS, and JavaScript.
Users can enter an amount and simulate a donation through an intuitive UI.

🚀 Features

- 🧭 Navigation (Home & About)
- 💎 Custom header with KivyMD-style card
- 💖 Donation input with validation
- 📱 Fully responsive (mobile-friendly)
- ⚡ No page reloads (SPA-like behavior)

- JavaScript (Vanilla JS)


🌐 Live Demo

Deployed on Vercel:
👉 https://for-diamond.vercel.app


⚙️ How It Works

1. User enters a donation amount
2. Clicks the 💖 Donate button
3. Input is validated
4. A confirmation message is displayed

«Note: This is a simulation only (no real payment processing yet)»


📱 Mobile Support

The site is fully responsive:

- Flexible layouts using Flexbox
- Media queries for small screens
- Touch-friendly inputs and buttons
